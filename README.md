Mz Cinema 🎬
​Mz Cinema is a modern, mobile-first web application for streaming movies and TV shows, powered by the TMDB API. It features a highly responsive design, multi-language support (English and Arabic), and a dark/light mode toggle.
​✨ Features
​Extensive Content Library: Browse trending movies, TV shows, and specialized categories like Anime, Bollywood, Turkish, and K-Drama.
​Built-in Streaming Player: Integrated video player with support for multiple server sources (VidSrc, VidLink, AutoEmbed, etc.).
​TV Show Navigation: Dedicated season and episode selectors with "Next Episode" functionality.
​Multi-Language & RTL Support: Fully localized for Arabic with right-to-left (RTL) layout adjustments and the Cairo font.
​Personalization:
​My List: Save favorite titles to a local watchlist.
​History: Automatically tracks recently watched content and progress.
​Advanced Search & Filters: Search for titles or actors with filtering options for genres, years, and sorting.
​PWA Ready: Includes a web manifest for installation as a standalone app on mobile devices.
​Theming: Seamlessly switch between Dark and Light interface modes.
